Have you been struggling with how to install Indigo on your device? Look no further as we provide a step-by-step guide on how to install Indigo, a popular add-on for Kodi users.

The Struggle to Install Indigo

You’re not alone if you have experienced difficulty with installing Indigo. Many users face the challenge of installing Indigo due to its dependency on Fusion repository, which has been removed from the internet. As a result, many tutorials are now outdated and offer no help to users who want to install Indigo.

How to Install Indigo

Here are the steps to follow to successfully install Indigo on your device:

  1. Launch Kodi and click on the gear icon to go to settings
  2. Select File Manager
  3. Click on Add source and type this URL exacty as it is - https://nitronation.github.io/ 5. Name the source “Nitron Nation” and click OK
  4. Go back to the Kodi Homepage and click on Add-ons
  5. Select the Package Installer icon at the top left corner
  6. Click on Install from zip file and select Nitron Nation
  7. Choose repository.nitronation-3.2.zip and wait for it to install
  8. Go back to the same menu and click on Install from repository
  9. Select Nitron Repo and then Program add-ons
  10. Click on Indigo and then Install
  11. Wait for Indigo to install and then click on Open

With these simple steps, you’ve successfully installed Indigo on your device.

Summary of How to Install Indigo

To summarize the main points, the key is to access Nitron Nation as the new repository for Indigo and install it from there. This new method works around the challenge of accessing the no-longer available Fusion repository.

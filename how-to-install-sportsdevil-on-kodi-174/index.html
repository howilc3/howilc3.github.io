How to Install SportsDevil on Kodi 17.4

Installing SportsDevil on Kodi 17.4 becomes effortless when following the proper steps. First, visit the Kodi home screen and click on the “Settings” gear icon. Then, navigate to “File Manager” and add the repo source URL using the following steps:

  1. Click on “Add source” and then select “None”
  2. Type in the repository URL and name the repository
    (http://kdil.co/repo/)
  3. Go back to the main Kodi screen. Select Add-ons and open the package installer icon in the top left corner
  4. Click on “Install from zip file” and select the repo that you just created
  5. After opening the repo, click on the “repository.kodil.zip” file to install it
    Note: Wait a few seconds for the “Add-on enabled” notification
  6. Click on “Install from repository” and select the “Kodil Repository”
  7. Open the “Video Add-ons” folder and select “SportsDevil” for installation
  8. Wait for the “Add-on enabled” notification to appear
  9. You have successfully installed SportsDevil on Kodi 17.4!

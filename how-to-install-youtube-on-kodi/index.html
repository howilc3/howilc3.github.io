How to Install YouTube on Kodi

To install YouTube on your Kodi device, follow these simple steps:

  1. Open Kodi on your device.
  2. Go to “Add-ons” in the left-hand menu.
  3. Select the “Package Installer” icon on the top left of the screen.
  4. Select “Install from repository.”
  5. Select “Kodi Add-on repository.”
  6. Scroll down until you see “Video add-ons” and select it.
  7. Find the “YouTube” add-on and select it.
  8. Press “Install” and wait for the add-on to install successfully.
  9. You’re all set! Navigate to “Add-ons” on Kodi and select the YouTube add-on to start watching videos.

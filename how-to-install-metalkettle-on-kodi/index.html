How to Install Metalkettle on Kodi

The process of installing Metalkettle is relatively simple and straightforward. However, before you begin, you’ll have to allow the installation of third-party repositories and add-ons on Kodi. To do this, navigate to the “Settings” option from the Kodi homepage, then select “System Settings.” From there, click on “Add-ons,” then toggle on the “Unknown Sources” option.

Next, you’ll need to obtain the Metalkettle repository file. You can find this on the internet from various sources. Once you’ve downloaded the file, navigate to the “Add-ons” section from the Kodi home screen, then select “Package Installer” (it looks like a box). From there, choose “Install from zip file,” then navigate to the directory where you saved the Metalkettle repository file. Once you’ve selected the file, the repository will install automatically.

Now that you’ve installed the Metalkettle repository, you can add it to your Kodi addons. To do this, navigate to “Install from repository,” then select “MetalKettles Addon Repository.” From here, you can select and install any addons available!

If you’re having trouble installing the Metalkettle repository, it’s likely that you haven’t enabled installation of unknown sources. Go back to the “System Settings” section from the Kodi homepage and ensure that the “Unknown Sources” option is toggled on.

How to Install Kodi using Downloader

The Downloader app is a safe, straightforward way to install Kodi on your Firestick. Here’s how to do it:

  1. Open the Downloader app and navigate to the settings menu.
  2. Enable “JavaScript” if it is not already enabled.
  3. Go back to the main menu and enter the URL of the Kodi installation file. This will typically end in “.apk”.
  4. Click “Download” and wait for the Kodi file to download onto your device.
  5. Open the Kodi file and click “Install”.

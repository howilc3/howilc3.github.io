How to Install Bob Unleashed

The first step towards installing Bob Unleashed is to enable unknown sources for the Kodi application. This is a necessary step because Bob Unleashed is a third-party add-on. To enable unknown sources, click on the Settings gear icon at the top left corner in Kodi’s main menu. Then click on “System settings”. Find and select the “Add-ons” option. Then, turn on the “Unknown sources” option. You’ll have to provide a confirmation for this.

Next, you need to download the Bob Unleashed repository. Go to Kodi’s main menu, click on “Add-ons,” and then click on the box icon located at the top left corner. Select “Install from zip file.” A popup menu will appear. Locate the folder where you downloaded the Bob Unleashed repository and select it. Click on the “OK” button.

Install the Bob Unleashed add-on

Now that you have successfully added the Bob Unleashed repository, you need to navigate to the “Install from repository” option and install the Bob Unleashed add-on. Go to the Kodi main menu and click on “Add-ons”. Then click on the box icon at the top left corner and select “Install from repository”. From there, search for the Bob Unleashed repository, click on it, and select “Video add-ons.” Finally, click on the “Bob Unleashed” add-on and press the “Install” button.

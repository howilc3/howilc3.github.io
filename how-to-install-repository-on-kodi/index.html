How to Install a Repository on Kodi?

Installing a repository on Kodi is quite simple, but you must have a few plugins installed before you begin. If you already have these plugins installed, then you’re ready to start. Follow these steps to install a repository on Kodi:

Step 1: Go to Kodi’s home screen and click on the “Settings” icon at the top left of the screen.

Step 2: Click on “System” - the icon looks like a gear icon.

Step 3: Click on the “Add-ons” submenu. On the right-hand side of the screen, enable the toggle switch that says “Unknown sources”.

Step 4: Click on “Yes” when the warning message appears.

Step 5: Go back to the Kodi home screen and click on “Settings” again. Then, click on “File manager”.

Step 6: Click on “Add source”.

Step 7: Click on "<None>" and add the URL of the repository source. Then, type in a name for the repository in the “Enter a name for this media source” field.

Step 8: Go back to the Kodi home screen and click on “Add-ons”.

Step 9: Click on the open box icon at the top left of the screen and select “Install from zip file”.

Step 10: Select the name of the repository from the list and click on the zip file.

Step 11: Wait for a few seconds for the repository to install. You will get a notification when the installation is complete.

How to install custom zombie maps waw

The first step is to download the custom zombie map files from a trusted source. We recommend downloading from reputable sites like the official Call of Duty forums or the Steam community.

Once you’ve downloaded the files, extract them to your game’s “mods” folder. The default location is C:\Program Files (x86)\Activision\Call of Duty - World at War\mods.

Next, launch World at War and enable the mod under the “mods” menu. Finally, start a new game and select the custom zombie map you want to play.
